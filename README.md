## 各自環境での導入方法

1. Node.JSのインストール  
  https://nodejs.org/ja/ からインストールする

1. Git cloneしてソース落としてくる。  
   該当ディレクトリに移動

1. npmアップデート  
   `npm install -g npm`

1. npm packageをインストール  
   `npm install`

1. Webサーバを起動
   `npm start`

1. localhost ポート3000番でアクセス
